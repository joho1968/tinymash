# Tinymash Identity Assets

This directory holds the default images that ship with tinymash itself.

They are the fallback identity files for a new installation:

- `favicon_ico.ico`
  Browser favicon.
- `tinymash_ico.png`
  Square tinymash icon.
- `tinymash_og.png`
  Default social-sharing image.
- `tinymash_banner.png`
  Default wide banner image.
- `missing-imported-media.svg`
  Fallback image used when an imported source file can no longer be recovered.

Themes may choose to use these files, replace them, or ignore them, but they are part of the shipped product rather than a specific theme.
